# Sephora Customer Review Analysis Dashboard

## Project Overview

This project analyzes Sephora customer reviews and product information using Python and Power BI. The goal was to explore customer satisfaction, identify the most reviewed brands, and visualize key insights through an interactive dashboard.

## Tools Used

* Python
* Pandas
* Matplotlib
* Power BI

## Dataset

Sephora Products & Reviews Dataset from Kaggle.

## Key Findings

* Analyzed over 602,000 customer reviews.
* Average customer rating: 4.31/5.
* Approximately 89% of reviews were positive.
* Tatcha was the most reviewed brand.
* 5-star ratings dominated customer feedback.

## Dashboard Features

* Total Reviews KPI
* Average Rating KPI
* Customer Sentiment Overview
* Top 10 Brands by Review Volume
* Rating Distribution Analysis
